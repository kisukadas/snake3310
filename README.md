A faithful recreation of the classic Nokia Snake game, built with vanilla HTML, CSS, and JavaScript.

## Features

- **Retro Aesthetics**: Styled to look like a classic Nokia phone.
- **CRT Effect**: Custom CSS scanlines and RGB sub-pixel rendering for a realistic screen look.
- **High Score**: Saves your high score locally.
- **Responsive Controls**: Play with arrow keys or the on-screen D-pad.

## How to Play

1. Open `index.html` in your web browser.
2. Use the **Arrow Keys** to move the snake.
3. Eat the food to grow and earn points.
4. Don't hit the walls or yourself!

## Development

No build step required! Just edit the files and refresh your browser.
